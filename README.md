# 18 宠物管理系统 java518

## 项目概述

18 宠物管理系统是一款基于 Spring Boot 和 Vue 的前后端分离系统。系统致力于提供宠物管理的一站式解决方案，包含用户端和管理员端，功能全面，易于拓展。

## 技术栈

### 后端技术

- Spring Boot
- MyBatis

### 前端技术

- Vue
- ElementUI
- Axios
- EChart
- HTML
- JavaScript
- CSS

### 系统环境

- IDEA
- JDK 1.8
- MySQL
- Maven

## 功能模块

### 用户端

- 宠物喂养
- 流浪宠物救助
- 宠物喂养点
- 丢失宠物查看
- 流浪宠物救助站
- 宠物论坛
- 宠物捐赠
- 科普文章
- 热门活动

### 管理员端

- 流浪动物管理
- 申请领养管理
- 评论管理
- 流浪动物救助管理
- 喂养点管理
- 动物走失管理
- 救助站管理
- 帖子管理
- 捐赠管理
- 公告管理
- 科普文章管理
- 活动管理

## 系统角色

系统分为用户端角色和管理员端角色，分别拥有不同的操作权限和功能模块。

## 运行环境

系统可在以下环境直接运行：

- IDEA 开发工具
- JDK 1.8
- MySQL 数据库
- Maven 项目构建工具

## 部署步骤

1. 确保环境配置正确，包括 JDK、MySQL 和 Maven。
2. 导入项目源码至 IDEA。
3. 配置数据库连接信息。
4. 运行 Maven 命令进行项目编译和打包。
5. 部署至服务器。
